Đồng bộ nhiều máy (GitHub Gist)

Dữ liệu chung được lưu trong một Gist bí mật trên GitHub. Mỗi máy (bạn và bạn cùng nhà) dán cùng một Token + Gist ID vào đây một lần — sau đó ai nhập gì, máy kia tự thấy. Token/API key chỉ nằm trên từng máy, không bao giờ bị đồng bộ.

Hướng dẫn cho cả nhà (bấm để xem)
  1. Chủ nhà (làm 1 lần): bấm link tạo token ở trên (Expiration chọn No expiration, chỉ tick gist) → dán token vào ô → bấm ➕ Tạo mới để có Gist ID → bấm Lưu cài đặt.
  2. Gửi riêng cho bạn cùng nhà đúng 2 thứ: Token và Gist ID (nhắn tin riêng, đừng đăng công khai).
  3. Bạn cùng nhà: mở trang web → ⚙️ → dán Token + Gist ID → Lưu cài đặt. Xong.
⚠ Token này cho phép đọc/ghi mọi Gist của tài khoản tạo nó — chỉ đưa cho người tin tưởng, hoặc tạo một tài khoản GitHub mới riêng cho việc này. Nếu lỡ lộ: vào GitHub → Settings → Tokens → Delete là vô hiệu ngay.

🤖 Scan hoá đơn bằng AI

Chọn file PDF gộp các hoá đơn đã scan (hoặc ảnh chụp). Trên hoá đơn bạn có thể note tay: tên/chữ cái đầu của người có phần riêng (P, L, B…), “/2” nếu chia 2, ai trả tiền… AI sẽ đọc và đề xuất danh sách hoá đơn — bạn luôn được xem lại và sửa trước khi thêm vào bảng.

💰 Chi phí ước tính: khoảng $0.01–$0.08 cho mỗi trang hoá đơn (tuỳ model). Chi phí thực tế hiển thị sau khi chạy.
